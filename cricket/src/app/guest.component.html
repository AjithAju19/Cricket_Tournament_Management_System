Cricket Player Management System

PROVIDING YOUNG CRICKETERS A PLACE TO FULFILL THEIR DREAMS




Objective

The aim of this project is to build a player management system that will completely automate the process of new player registration in a league. The system will handle the insertion, deletion, updating and display records of league players.


Existing System

The existing system requires many documents and application forms to be filled out. Data has to be repeatedly entered at each stage of the process and multiple files are made for each player.

Proposed System

The proposed system will eliminate the paper trail required for the completion of various formalities. The entire process of prospective player record till new player record can be handled by this system. The presence of this system will remove ambiguity in the process that can confuse the selectors.







What can be done by the proposed system



1. Insert a player details:


Once a prospective player achieves an acceptable rank to enter into the league. The selector will scrutinize the details of users in the insertion module and notify them about discrepancies. This module will automatically update the “Player Management” database with the details of the new player.

2. Update player details:


This module helps in updating the player details as the game progress, the league administrator would do so after each and every match and keep track of player records.

3. Display player details:


This module helps in displaying the player records by retrieving from the database. The details like player name, runs scored, rating, age would be listed.







4. Delete player details:


This module is to delete / remove a player from the league i.e., deleting his records from the database. To do so type the player name and Click “DELETE”, the player record would be deleted.

5.Gallery


Provides room for users to watch match highlights and view ICC rankings published till date.

6.Managing Teams


Once a prospective team achieves an acceptable rank to enter into the league. The selector will scrutinize the details of users in the insertion module and notify them about discrepancies. This module will automatically update the “Player Management” database with the details of the new team.






About ICC



The ICC is the global governing body for cricket. Representing 106 members, the ICC governs and administrates the game and works with our members to grow the sport. The ICC is also responsible for the staging of all ICC Events. The ICC presides over the ICC Code of Conduct, playing conditions, the Decision Review System and other ICC regulations. The ICC also appoints all match officials that officiate at all sanctioned international matches. Through the Anti-Corruption Unit it coordinates action against corruption and match fixing.
